# 🎬 MongoDB Movie Review Project

## 📌 Description
This project connects MongoDB with Python using PyMongo and performs data analysis on movie reviews.

## 🛠 Technologies Used
- MongoDB
- Python
- Pandas
- PyMongo

## 📂 Project Structure
movie-review-project/
│── main.py
│── requirements.txt
│── README.md

## ▶️ How to Run

1. Start MongoDB
2. Insert data into `movie_reviews.movies`
3. Run:
   python main.py

## 📊 Features
- Fetch data from MongoDB
- Convert to DataFrame
- Process nested data (reviews)
- Display structured output
